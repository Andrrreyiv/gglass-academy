# Gglass Academy — лендинг

Статическая версия сайта обучения ремонту автостёкол (перенос с Lovable).
Один экран, без сборки: чистый HTML + CSS + vanilla JS + PHP-приёмник формы.

## Структура

```
index.html          — главная страница (разметка 1:1 с оригиналом)
privacy.html        — политика обработки ПДн (152-ФЗ)
favicon.svg
assets/
  styles.css        — оригинальные стили (Tailwind-сборка Lovable), не трогаем
  enhance.css       — слой улучшений: шрифты, контраст WCAG AA, анимации, состояния формы
  app.js            — форма заявки, маска телефона, scroll-reveal, FAQ
  fonts/            — self-hosted woff2 (Inter / Unbounded / JetBrains Mono, cyr+lat)
  *.webp            — изображения
api/
  lead.php          — приём заявки: письмо на почту + лог в api/leads/ (в .gitignore)
```

## Что улучшено относительно оригинала

- **Шрифты.** В оригинале Unbounded / Inter / JetBrains Mono были объявлены в CSS,
  но не загружались (`@font-face` отсутствовал) — пользователи видели системный шрифт.
  Теперь шрифты self-hosted (woff2, кириллица+латиница), `font-display:swap`.
- **Контраст WCAG AA.** Тон `--muted` слегка поднят, чтобы opacity-варианты текста
  проходили порог 4.5:1.
- **Микроанимации.** Плавное появление секций при скролле (IntersectionObserver),
  hover-подъём кнопок и карточек. Всё отключается при `prefers-reduced-motion`.
- **Форма и 152-ФЗ.** Добавлен обязательный, НЕ предустановленный чекбокс согласия
  на обработку ПДн со ссылкой на политику; маска телефона; состояния
  focus / error / loading / success; отправка на российский бэкенд (PHP).
- **Доступность.** Видимый `:focus-visible`, `scroll-margin` под sticky-хедер,
  корректный `lang="ru"`.
- **Чистка.** Удалены React/TanStack-бандлы и трекеры Lovable (`~flock.js`, `__l5e`),
  оставлена только Яндекс.Метрика (РФ-аналитика).

## Деплой (Phase 2 — российский хостинг)

Требуется хостинг с поддержкой **PHP** (mail() или SMTP) на территории РФ.

1. Загрузить все файлы в корень сайта.
2. Проверить, что каталог `api/leads/` создаётся и доступен на запись (0750).
3. Настроить доставку почты. Для Яндекс.Почты надёжнее SMTP (логин/пароль приложения)
   вместо `mail()` — задать через переменные окружения:
   - `LEAD_TO` — получатель (по умолчанию `gglass-detailing@yandex.ru`);
   - `LEAD_FROM` — адрес отправителя на домене (напр. `no-reply@gglass-academy.ru`).
4. Привязать домен: сейчас работает только `www.gglass-academy.ru`,
   apex `gglass-academy.ru` отдаёт 421 — настроить DNS/редирект apex → www.
5. Включить HTTPS (обязательно для передачи ПДн).
6. Клиенту: подать уведомление в Роскомнадзор об обработке ПДн (актуально с 30.05.2025).
